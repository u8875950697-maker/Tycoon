# Roadmap

## Milestone 1: Concept Validation
- Finalize art direction references (paper-style inspiration, color palette).
- Outline key gameplay loops and resource flow diagrams.
- Produce early UI wireframes for core screens.

## Milestone 2: Prototype Foundations
- Choose engine/tools and establish project settings.
- Block out placeholder scenes for farm, biome selection, and menus.
- Implement mocked resource counters and manual caretaking interactions.

## Milestone 3: Vertical Slice
- Add representative tree types with unique visuals and simple growth curves.
- Integrate baseline audio and VFX cues for feedback.
- Prepare build + testing checklist for internal feedback.

## Milestone 4: Pre-Alpha
- Expand automation systems, prestige loop, and event schedule specs.
- Flesh out content pipelines for art, UI, and localization.
- Set up analytics and QA plans before broader playtesting.
